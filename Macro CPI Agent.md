這份 **Macro CPI Agent v2.2** 已經不只是一個顯示數據的儀表板，它是一套**「模擬避險基金思維」**的自動化分析系統。

它的設計核心邏輯是：**從上游到下游、從結構到預期、從過去到未來**。

以下我為你完整拆解這個 Agent 的**五大分析模組**與**實戰功能**，讓你清楚知道每一張圖表背後的投資意義：

---

### 1. 頂層戰情看板 (The KPI Board)
這是你打開網頁第一眼看到的四個數字，用來判斷**「當下的宏觀溫度」**。

| 指標 | 意義 | 投資解讀 |
| :--- | :--- | :--- |
| **CPI (Headline)** | 整體通膨 | 告訴你生活成本漲多少。若數值 > 3%，Fed 降息機率低。 |
| **PPI (Final Demand)** | 生產成本 | 工廠出貨價格。**它是 CPI 的領先指標**。 |
| **Supercore** | **超級核心** | **最重要指標**（服務扣除房租）。這是 Fed 主席鮑爾（Powell）決定是否降息的關鍵。若此數值不降，利率就不會降。 |
| **利潤剪刀差** | **企業獲利** | `CPI - PPI`。數字越大，代表企業賣得貴、成本低，獲利強勁（股市利多）。 |

---

### 2. 五大深度分析模組 (Deep Dive Tabs)

這五個分頁代表了五個不同的分析維度：

#### 📊 Tab 1: 核心趨勢 (傳導鏈分析)
*   **分析邏輯**：物價是怎麼傳導的？
*   **圖表 1 (CPI vs PPI)**：
    *   觀察 **PPI (藍線)** 是否領先 **CPI (紅線)** 下跌？
    *   *觀點*：如果 PPI 已經崩盤但 CPI 還很高，代表通膨幾個月後就會下來（買債機會）。
*   **圖表 2 (剪刀差 Profit Spread)**：
    *   **正值區 ( > 0)**：企業有定價權，成本轉嫁給消費者 -> **買股票 (尤其消費股)**。
    *   **負值區 ( < 0)**：成本漲太兇，企業不敢漲價 -> **避開低毛利製造業**。

#### 🏗️ Tab 2: 結構拆解 (魔鬼藏在細節裡)
*   **分析邏輯**：通膨是暫時的（油價）還是永久的（薪資）？
*   **圖表 1 (Sticky vs Core)**：
    *   **Sticky CPI (黏性通膨)**：理髮、醫療、房租。這些價格一旦漲上去就不回頭。
    *   *觀點*：只要紅線 (Sticky) 還在高檔，就算油價跌，通膨也只是「假摔」。
*   **圖表 2 (Used Cars - 領先指標)**：
    *   **二手車**：它是核心商品通膨的「金絲雀」。
    *   *觀點*：二手車價通常**領先核心 CPI 約 2-3 個月**。看到綠線跳水，就大膽預測下季通膨會降。
*   **圖表 3 (Supercore - 魔王關)**：
    *   這是 Fed 最怕的「薪資螺旋」。這條線沒下來，Fed 絕對不敢寬鬆。

#### 🏭 Tab 3: 全球供應鏈 (輸入性通膨)
*   **分析邏輯**：通膨是美國自己印鈔搞的，還是外國漲價傳過來的？
*   **關鍵指標**：**中國進口價格 (China Import Prices)**。
*   **實戰應用**：
    *   中國是世界工廠。如果這條線是 **負值 ( < 0 )**，代表中國正在「輸出通縮」（降價賣貨）。
    *   這對美國通膨是**大利多**（可以抵銷美國內部的服務業通膨）。

#### 🧠 Tab 4: 市場預期 (交易員在賭什麼)
*   **分析邏輯**：現在的股價/債價，是否已經反映了未來的通膨？
*   **圖表 1 (實質利率 Real Yield)**：
    *   公式：`名目利率 - 通膨預期`。這是資金的真實成本。
    *   *警戒線*：**> 2.0%**。資金成本極高，對**科技成長股 (Nasdaq)** 是殺手。
*   **圖表 2 (Breakeven - 平衡通膨率)**：
    *   債券市場「賭」出來的未來通膨。
    *   *觀點*：如果媒體大喊通膨失控，但這條線卻很平穩，代表市場認為「媒體在嚇人」，你可以反向操作。

#### 🔮 Tab 5: 未來模擬器 (數學預測)
*   **這是本 Agent 最強大的殺手級功能。**
*   **原理**：**基期效應 (Base Effect)**。去年的 CPI 數字（分母）已經固定了，我們可以推算出未來的「數學路徑」。
*   **怎麼用？**
    *   拉動滑桿設定：假設未來每個月物價微漲 0.2% (MoM)。
    *   **看結果**：圖表會畫出未來 6 個月的紅線。
    *   *價值*：你會發現，「喔！原來下個月通膨**必定**會反彈，因為去年的基期太低了！」
    *   這讓你比看新聞的人**早一個月知道**下個月 CPI 數據是利多還是利空。

### 3. 教育功能 (Education Feature)

這是為了防止你忘記上述邏輯而設計的：
*   在每一張圖表下方，都有一個 **📖 點我展開** 的按鈕。
*   點開後，會有：
    1.  **🧐 定義**：這是什麼？
    2.  **🍎 舉例**：用雞排、麵包店舉例。
    3.  **⚡ 訊號**：看到這個數據，該買還是該賣？

### 總結：如何使用這個 Agent 賺錢/避險？

1.  **每月 CPI 公布前**：打開 **Tab 5 (模擬器)**，看看下個月數據是否會因為基期效應而「暴雷」。
2.  **平常日**：看 **Tab 4 (實質利率)**，如果實質利率衝破 2%，減碼高估值科技股。
3.  **選股時**：看 **Tab 1 (剪刀差)**，如果剪刀差擴大，大力買進有品牌的消費股（如 Costco, Apple），因為他們成本降了但沒降價，利潤會暴增。
4.  **判斷 Fed 轉向**：死盯著 **Tab 2 的 Supercore**，這條線不崩，別輕易相信 Fed 會大幅降息。
